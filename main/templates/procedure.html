Instructions for Filling Event Request Form

This page provides step-by-step instructions for filling out the Event Request Form. Please follow these guidelines to ensure your event request is processed smoothly.

Step 1: Halls Required

Enter the room numbers of the halls you require for your event. Click the "Add Hall" button to add additional halls if needed.

Step 2: Event Name

Provide a name for your event in the "Name of the event" field.

Step 3: Date(s) of the Programme

Add the date(s) of your event program. For each date, select the date using the calendar input and specify the start and end times for that date. Click the "Add Date" button to add more dates if necessary.

Step 4: Purpose

Describe the purpose of your event in detail in the "Purpose (long description)" field.

Step 5: Audio System and Projector

Check the appropriate checkboxes if you require an Audio System and/or a Projector for your event.

Step 6: Event Coordinator Details

Provide the details of the event coordinator, including their name, email address (must be in the iiitt.ac.in domain), and select their department from the dropdown menu.

Step 7: Undertaking

Check the "Undertaking" checkbox to indicate that you take responsibility for taking care of all electronic and non-electronic items during the event.

Step 8: Upload Sanction Letter

Upload a PDF copy of the sanction letter by clicking the "Choose File" button and selecting the appropriate file. The uploaded file must be in PDF format.

Step 9: Submit

After completing all the sections of the form, click the "Submit" button to send your event request.

If you have any questions or need assistance, please contact the event organizers or the appropriate department.
